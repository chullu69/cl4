 - The dataset below (copy and paste into Excel)

Steps:
1. Open a new Excel workbook.
2. Paste the dataset starting at cell A1. It will fill A1:E14 (including
   the totals row). Adjust column widths if needed.
3. Format the header row (bold).

----------------------------------------------------------------------
BASIC CHARTS
----------------------------------------------------------------------
A. Bar Chart
   - Select range A1:D13 (exclude totals row).
   - Insert → Charts → Bar → 2‑D Bar → Clustered Bar.
   - Chart will compare monthly sales of TVs, Mobile Phones, Laptops.

B. Column Chart
   - Same selection A1:D13.
   - Insert → Charts → Column → 2‑D Column → Clustered Column.

C. Line Chart
   - Same selection A1:D13.
   - Insert → Charts → Line → 2‑D Line → Line with Markers.

D. Pie Chart (Total Sales by Product)
   - Hold Ctrl and select B1:D1 (headers "TVs", "Mobile Phones", "Laptops").
   - Then hold Ctrl and select B14:D14 (the totals row).
   - Insert → Charts → Pie → 2‑D Pie → Pie.
   - Shows proportion of total sales across the three product categories.

----------------------------------------------------------------------
ADVANCED CHARTS
----------------------------------------------------------------------
E. Scatter Plot (TVs vs. Laptops)
   - Select range A1:C13 (Month, TVs, Laptops).
   - Insert → Charts → Scatter → Scatter with Smooth Lines and Markers.
   - Visualises correlation between TVs and Laptops sold.

F. Waterfall Chart (Monthly Totals)
   - Hold Ctrl and select A2:A13 (all month cells, excluding header).
   - Then hold Ctrl and select E2:E13 (all monthly Total values).
   - Insert → Charts → Waterfall → Waterfall.
   - Shows the cumulative total across months.

----------------------------------------------------------------------
CUSTOMIZATION (apply to any chart)
----------------------------------------------------------------------
- Add Chart Title:
  Chart Design → Add Chart Element → Chart Title → Above Chart.
  Type "Electronic Store Sales 2022".

- Move Legend:
  Chart Design → Add Chart Element → Legend → Top (or Right).

- Add Data Labels:
  Chart Design → Add Chart Element → Data Labels → Center (or Above).

- Change Style & Colors:
  Chart Design → choose a style from the gallery.
  Chart Design → Change Colors → select a monochromatic palette.

- Add Axis Titles:
  Chart Design → Add Chart Element → Axis Titles → Primary Horizontal
  (type "Month") and Primary Vertical (type "Units Sold").

- Format Axis (if needed):
  Right‑click on axis → Format Axis → adjust scale or number format.

----------------------------------------------------------------------
Save your workbook as "AdvancedExcel_Visualization.xlsx".
